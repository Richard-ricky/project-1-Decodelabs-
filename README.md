TechNova - Responsive Product Landing Page
Project Overview
TechNova is a responsive product landing page built using HTML5, CSS3, and JavaScript. The project demonstrates modern frontend development principles, including responsive design, accessibility, semantic HTML, CSS Grid, and Flexbox layouts.
This project was developed as part of the DecodeLabs Full Stack Development Project 1 assignment.
Features
Responsive design for mobile, tablet, and desktop devices
Semantic HTML5 structure
CSS Grid layout for feature and testimonial sections
Flexbox for navigation and pricing components
Dark/Light mode toggle
Interactive contact form
Modern and clean user interface
Accessibility-friendly design
Technologies Used
HTML5
CSS3
JavaScript (ES6)
